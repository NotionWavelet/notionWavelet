Preguntas frecuentes

Todo lo que necesitas saber antes de empezar

Resolvemos las dudas más habituales sobre instalación, funcionamiento, datos y soporte.

¿Qué incluye la demostración?

Te enseñamos el flujo de clientes, vehículos, órdenes, presupuestos y facturación con un caso práctico. Después decides si quieres contratar, sin compromiso.

¿Wavelet funciona con VeriFactu?

Wavelet genera hash fiscal, encadenamiento, QR, auditoría y archivo legal. La transmisión automática de facturas a la AEAT no se anuncia todavía como una función plenamente operativa.

¿Envía mensajes automáticamente por WhatsApp?

No. Wavelet guarda el PDF, abre WhatsApp Web y prepara el mensaje. El usuario revisa, adjunta y confirma el envío, ya que WhatsApp no permite automatizar libremente ese paso.

¿Puedo importar mis datos actuales?

Sí. Existen importaciones desde CSV y desde bases de datos compatibles para clientes, vehículos, órdenes, facturas, históricos y varios catálogos.

¿La migración tiene coste?

La migración inicial se ofrece gratuitamente tras revisar el formato, el volumen y la calidad de los datos disponibles.

¿Necesito conexión permanente a Internet?

La aplicación principal trabaja de forma local en Windows. Algunas acciones externas, como abrir WhatsApp Web o servicios online, sí necesitan conexión.

¿Funciona en móvil o tablet?

No como aplicación principal. Wavelet está diseñado actualmente para ordenadores con Windows 10 y Windows 11.

¿Tiene agenda de citas?

No. La versión actual no incluye un calendario de citas, recordatorios o planificación de disponibilidad.

¿Incluye control de stock completo?

Incluye catálogo de materiales, precios y unidades, pero todavía no debe considerarse un sistema completo de almacén con entradas, salidas, mínimos y compras.

¿Tiene TPV o cobro con tarjeta?

No. Permite registrar cobros manualmente, pero no integra terminal de tarjeta, caja ni pasarela de pago.

¿Incluye inteligencia artificial?

No. Wavelet no incorpora actualmente IA, asistentes generativos ni OCR inteligente.

¿Hace copias de seguridad?

Sí. Genera copias automáticas al cerrar la aplicación y después de emitir una factura, además de permitir comprobar y restaurar backups.

¿Puedo exportar información?

Puedes guardar documentos en PDF, exportar registros legales en JSON y conservar copias de seguridad de la base de datos.

¿Puedo cancelar cuando quiera?

Sí. El plan se factura mensualmente y no tiene permanencia.

¿Cuántos usuarios incluye?

El plan actual está orientado a un usuario por taller y a negocios pequeños donde una persona gestiona la parte administrativa.

¿Qué ocurre con una orden ya facturada?

La orden queda bloqueada para impedir cambios o una segunda facturación accidental.

¿Cómo se realiza la demostración?

Concertamos una sesión para enseñarte el programa con un flujo práctico y resolver las dudas de tu taller antes de contratar.

¿El precio es por usuario?

Sí. El Plan Taller Esencial incluye una licencia para un usuario por 49 €/mes + IVA.

¿Hay costes de alta?

No se anuncian costes de alta ni módulos obligatorios. La migración inicial compatible y la ayuda de puesta en marcha están incluidas.

¿Cómo recibo las actualizaciones?

Las mejoras y actualizaciones legales forman parte de la suscripción. El soporte te indicará el proceso de instalación cuando haya una nueva versión.

¿Qué ocurre si cambio de ordenador?

El cambio requiere instalar Wavelet en el nuevo equipo y trasladar una copia validada de la base de datos. El soporte puede orientarte durante el proceso.

¿Puedo imprimir presupuestos, órdenes y facturas?

Sí. Los documentos se pueden generar en PDF, previsualizar, guardar e imprimir desde la aplicación.

¿Mis datos se guardan en la nube?

No como base principal. Wavelet trabaja localmente en el ordenador del taller y genera copias de seguridad para proteger la información.

¿Puedo solicitar una función que todavía no existe?

Puedes comunicar necesidades y sugerencias al soporte. Su incorporación dependerá de la viabilidad y de la planificación del producto.
