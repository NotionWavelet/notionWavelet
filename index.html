¿Funciona sin internet?

Sí. La gestión diaria utiliza una base de datos local. Internet solo se necesita para funciones externas como correo o servicios fiscales conectados.

¿Dónde se guardan mis datos?

En el equipo Windows del taller, dentro de una base de datos SQLite local. También se generan copias automáticas.

¿Incluye VeriFactu?

Sí. La facturación incorpora QR, hash, encadenamiento, trazabilidad, archivo legal y gestión de anulaciones.

¿Puedo importar mis clientes y vehículos?

Sí. Puedes importar información desde archivos CSV y también migrar datos desde una base anterior compatible.

¿Genera presupuestos y PDF?

Sí. Desde la orden puedes generar presupuestos, resguardos, albaranes y facturas, previsualizarlos, imprimirlos o guardarlos en PDF.

¿Controla facturas pendientes de cobro?

Sí. Puedes registrar pagos, consultar importes pendientes y utilizar métodos de pago configurables.

¿Es para varios usuarios?

El plan está pensado para un usuario por taller y equipos pequeños de entre uno y tres empleados.

¿Necesito instalar otro programa o servidor?

No necesitas instalar un servidor de base de datos. Notion Wavelet funciona como aplicación de escritorio para Windows.

¿Hay permanencia?

No. La facturación es mensual y puedes cancelar sin penalización.

¿Qué incluye la prueba?

Durante 14 días puedes conocer el flujo completo y comprobar cómo encaja en el trabajo diario de tu taller, sin tarjeta.
