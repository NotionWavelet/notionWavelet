2. Objeto del sitio web

Este sitio ofrece información comercial sobre Notion Wavelet, una aplicación de gestión para talleres, y permite realizar consultas, solicitar una demostración o pedir acceso a una prueba.

La navegación por el sitio atribuye la condición de usuario e implica la aceptación de las condiciones publicadas en cada momento.

3. Uso correcto de la web

El usuario se compromete a utilizar el sitio de forma lícita, diligente y respetuosa. No está permitido:

  • Intentar acceder sin autorización a sistemas, áreas privadas o datos.
  • Introducir código malicioso o realizar acciones que afecten a la disponibilidad del servicio.
  • Usar los formularios para comunicaciones ilícitas, engañosas o masivas.
  • Suplantar a terceros o facilitar datos falsos deliberadamente.
